# Desktop Sync Steps

Manual steps to sync your desktop dotfiles after the laptop split.
Created: 2026-01-28

---

## Pre-Sync Checklist

Before making any changes, back up your current desktop state:

```fish
# Optional: backup current ~/.config state
cd ~/dotfiles
./backup_dotfiles.fish  # Select 1 for desktop
git stash  # or commit if you want to keep it
```

---

## 1. Create Missing machine.conf for Hyprland

Your `~/.config/hypr/machine.conf` doesn't exist. Create it from the desktop template:

```fish
cp ~/dotfiles/hypr/machines/desktop.conf ~/.config/hypr/machine.conf
```

**What this file contains** (desktop-specific settings):
- Monitor auto-configuration
- Discord with `--disable-gpu` flag

---

## 2. Add machine.conf Source to hyprland.conf

Your `~/.config/hypr/hyprland.conf` is missing the line that loads machine-specific config.

**Add this at the end of `~/.config/hypr/hyprland.conf`:**

```conf
# Machine-specific configuration
# This file is generated by restore_dotfiles.fish based on machine type
source=~/.config/hypr/machine.conf
```

---

## 3. Fix Waybar Config (Currently Hybrid)

Your `~/.config/waybar/config` has both laptop and desktop modules mixed. Replace with desktop-only version:

```fish
# Backup current hybrid config first
cp ~/.config/waybar/config ~/.config/waybar/config.backup

# Copy desktop config
cp ~/dotfiles/waybar/machines/desktop.json ~/.config/waybar/config
```

**Desktop waybar modules** (what you'll get):
- Left: workspaces, window title
- Right: docker, tray, cpu, memory, pulseaudio, clock
- No battery, brightness, or bluetooth (those are laptop-only)

---

## 4. Fix Foot Terminal Config

Your `~/.config/foot/foot.ini` has different theme colors (serene-nature) than the repo (cozy-cabin).

**Option A: Keep your current theme** - no action needed

**Option B: Use repo's desktop config:**
```fish
cp ~/dotfiles/foot/machines/desktop.ini ~/.config/foot/foot.ini
```

Desktop foot config has:
- Font size: 11
- dpi-aware: yes

---

## 5. Remove Hardcoded Discord Binding from keybindings.conf

The Discord keybinding is hardcoded in `~/.config/hypr/keybindings.conf` but should come from `machine.conf`.

**In `~/.config/hypr/keybindings.conf`, find and comment out (around line 87):**

```conf
# Comment out or remove this line:
bind = Super, D, exec, discord --enable-features=UseOzonePlatform,WebRTCPipeWireCapturer --ozone-platform=wayland --disable-gpu

# Replace with a comment noting it's in machine.conf:
# Note: Super+D (Discord) is defined in machine.conf (different flags per machine)
```

The Discord binding is already in `hypr/machines/desktop.conf`, so machine.conf will handle it.

---

## 6. Theme Mismatch (Optional)

Your ~/.config uses **serene-nature** theme, repo has **cozy-cabin**.

**If you want to keep serene-nature:** Update the repo with your current theme colors:
```fish
cd ~/dotfiles
./backup_dotfiles.fish  # Select 1 for desktop
# Then commit the theme changes
```

**If you want cozy-cabin back:** Run the restore script or manually copy:
```fish
cp ~/dotfiles/waybar/style.css ~/.config/waybar/style.css
cp ~/dotfiles/foot/foot.ini ~/.config/foot/foot.ini
# etc.
```

---

## 7. GTK_THEME Setting (Optional)

`GTK_THEME=Adwaita:dark` was removed from fish config but added to hyprland.conf as `env = GTK_THEME,Breeze-Dark`.

**Current state:** GTK theme is set in hyprland.conf, not fish. This is fine, just be aware.

**If you want it in fish instead**, add to `~/.config/fish/config.fish`:
```fish
set -x GTK_THEME Adwaita:dark
```

---

## 8. Reload Everything

After making changes:

```fish
# Reload hyprland
hyprctl reload

# Restart waybar
killall waybar; waybar &

# Restart foot (just open a new terminal)

# Reload fish config
source ~/.config/fish/config.fish
```

```fish

```
